# ItsTyne Consult

ItsTyne Consult is an open-source Flutter application that demonstrates
a request-based consultation booking system using Flutter and Firebase.

This project is built for Flutter learners and developers who want to
understand real-world app architecture, RBAC, and Firebase integration.

## Features
- Request-based consultation booking
- Role-Based Access Control (RBAC)
- Firebase Authentication (Email & Google)
- Firestore database
- Admin approval workflow
- Push notifications (FCM)
- Clean Flutter architecture
- Dark & Light mode

## User Roles
- SuperAdmin
- Admin
- Consultant
- Client
- Guest

## Tech Stack
- Flutter
- Firebase Auth
- Cloud Firestore
- Firebase Cloud Functions
- Firebase Cloud Messaging
